# 正式实验补充：oracle-only 口径（2026-07-04）

## 1. 实验设置

本轮按审稿可接受的 fixed-budget / oracle 口径重跑，不展示 wall-clock。

- 图像尺寸：128 x 128
- 测试集：64 张
- 评测步数：K = 1000
- 参考解：fixed plain FBHF 长跑 4000 步
- 数据：真实图、合成 TV 图、wide Gaussian blur + high noise、motion blur + high noise
- 主模型：`wbv_seed11.pt`
- motion 适配模型：`wbv_motion_seed11.pt`
- oracle 计数：prox + B + C；line-search 的回溯计入 oracle
- learned-warped 每步含 1 次 network forward，但本轮不把 wall-clock 写进图或结论

## 2. 主要 endpoint 结果

误差为相对 primal error，越小越好。

| 场景 | plain FBHF | line-search FBHF | fixed warped | learned warped | 结论 |
|---|---:|---:|---:|---:|---|
| real / Gaussian train | 7.60e-4 | 7.15e-4 | 2.69e-4 | **1.40e-4** | 比 fixed 低 48% |
| synthetic / Gaussian train | 8.27e-3 | 8.91e-3 | 1.49e-3 | **1.06e-3** | 比 fixed 低 29% |
| real / wide Gaussian + high noise | 2.44e-3 | 2.31e-3 | 3.94e-4 | **2.50e-4** | 比 fixed 低 37% |
| real / motion + high noise，Gaussian-trained | 3.16e-4 | **2.03e-4** | 7.28e-2 | 1.45e-2 | OOD 失败，不能作为泛化胜利 |
| real / motion + high noise，motion-trained | 3.16e-4 | 2.03e-4 | 7.28e-2 | **9.25e-5** | 分布适配后胜过 line-search |

核心结论：learned warped 预条件器在同一 oracle 预算下稳定优于 fixed warped；motion blur 不是无条件泛化成功，而是需要匹配训练分布。

## 3. c 扫描

真实图 Gaussian train，K = 1000。

| c | fixed warped | learned warped |
|---:|---:|---:|
| 0.25 | 2.60e-4 | **1.51e-4** |
| 0.50 | 2.60e-4 | **1.60e-4** |
| 0.90 | 2.60e-4 | **1.59e-4** |

结论：learned-warped 对 c 不敏感，0.25 最好但 0.5/0.9 也稳定优于 fixed warped。手稿可用 c=0.5 作主设置，c 扫描作稳健性图。

## 4. 该怎么写进论文

可以写：

> Under the same oracle budget, the learned variable-metric FBHF consistently improves fixed-budget reconstruction quality over a fixed warped metric. The gain persists on real images and stronger Gaussian blur. Motion blur exposes a distribution-shift failure for a Gaussian-trained model, while a motion-trained metric restores and even improves performance.

不能写：

> learned-warped wall-clock 更快。

原因：learned-warped 每步多一次网络前向，本轮故意不展示 wall-clock；结论只限定在 oracle / fixed-budget reconstruction quality。

## 6. 当前判断

B 轨不再是“实验完全没支撑”。更准确的定位是：

1. 可证贡献：warped / variable-metric FBHF 的结构安全性。
2. 实验贡献：同 oracle 预算下，学习度量比固定度量给出更好重建质量。
3. 边界条件：泛化不是自动成立，motion blur 需要训练分布覆盖；因此手稿不能把它写成无条件 OOD 泛化。
